    Descobrindo a versão do seu Nintendo Switch

    A forma mais prática e rápida de descobrir se a versão do seu Nintendo Switch é a versão em que a falha fusee-gelee foi corrida ou não, é pegando o serial number dele e verificando em algum site ou utilizando a lista abaixo que estamos disponibilizando abaixo, os dois sites não possuem diferenças práticas, os dois utilizan essa mesma lista como base para definir se seu Switch é corrigido ou não.

    Sites de verificação de serial number

    A forma de se verificar é utilizando um desses sites abaixo, fica a seu critério:

    • ismyswitchpatched
    • ssnc
